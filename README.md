UX Design Project

Dit project is een webtoepassing gericht op het verbeteren van de klaservaring. Het biedt interactieve tools waarmee studenten beslissingen kunnen nemen, zoals het openen van een raam of het kiezen van een pauzetijd, zonder de les te onderbreken.

Functionaliteiten 

Stemsyteem: Gebruiksvriendelijke interface voor groepsbeslissingen, met live feedback en visuele grafieken.

Pauzeplanner: Selecteer eenvoudig een pauzetijd via een interactieve lijst.

Responsief ontwerp: Geoptimaliseerd voor zowel desktop als mobiel.

Technologieën:

HTML5

CSS3

JavaScript
